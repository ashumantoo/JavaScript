--> Event Loop 
    Phases
      - Timers Phase          -> setTimeOut and setIntervals
      - Pending Phase         -> Runs the pending callbacks from prev tick
      - Idle, Prepare Phase   -> internal to NodeJS
      - Poll Phase            -> Heart of the event loop - New I/O, Runs almost all the JS callbacks
      - Check Phase           -> setImmediate callbacks
      - Close Callbacks Phase -> Close connection callbacks
    * Priority from top to button
    * Microtasks Phase: It has high Priority than the event Phases
      - process.nextTick()
      - Promise
    * After every tick the microtask queue got drains completly
